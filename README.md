# Nabd App

تطبيق nabd – نظام تدفق وتطبيق وعي صحي متنقل لمشروع التخرج

## الوصف
تطبيق طبي شامل يساعد المستخدمين في:
- التحقق من الأعراض بالذكاء الاصطناعي
- حجز مواعيد مع أطباء
- خدمات الطوارئ والإسعاف
- بنك الدم والمستشفيات
- متابعة السجلات الطبية والصيدليات
- المتابعه الصحيه للمريض والرعاية الكامله خارج وداخل المستشفى

## التقنيات المستخدمة
- Flutter (Dart)
- Firebase (Auth, Firestore, Storage, Messaging)
- Bloc / Cubit لإدارة الحالة
- Stripe للدفع
- Agora للمكالمات الصوتية/الفيديو
- وغيرها...
